### Section 7: Outreach Sequences

```markdown
## 7. Contact Sequences

### Sequence A: Tier 5-6 IC Approach
**Goal:** Gather team intel, identify pain points

**Week 1:**
- LinkedIn connection request with personalized note
- Research their background/projects

**Week 2:**
- If connected: Send value-add message (industry article, job market insight)
- If not connected: Try email introduction

**Week 3:**
- Request informal call ("learning about the program")
- Prepare questions (see HUMINT skill)

### Sequence B: Tier 4 Manager Approach
**Goal:** Validate intel, discuss staffing needs

**Week 1:**
- Reference Tier 5-6 conversation: "Your colleague [Name] mentioned..."
- Offer specific value (candidate introduction, capability overview)

**Week 2:**
- Follow up with relevant content
- Request brief call to discuss their needs

**Week 3:**
- Present 2-3 candidates matching their requirements
- Ask about broader team needs

### Sequence C: Tier 3 PM Approach
**Goal:** Position for partnership/subcontract opportunity

**Week 1:**
- Executive intro from GDIT leadership if available
- Otherwise, reference validated pain points and solutions

**Week 2:**
- Request meeting: "I'd like to discuss how we can support [PROGRAM]"
- Prepare capability presentation

**Week 3:**
- Present tailored solution addressing specific pain points
- Propose next steps (teaming agreement, candidate submittals, etc.)
```

### Auto-Generation Triggers

1. **New Program Identified**: When Federal Programs DB gets new entry with high priority
2. **Hot Opportunity Signal**: When job scraping detects multiple posts at same site
3. **Contract Event**: When contract review/recompete date approaches (180 days out)
4. **Win Theme Update**: When new past performance is added
