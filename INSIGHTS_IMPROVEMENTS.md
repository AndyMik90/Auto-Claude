## 📝 הערות טכניות

### תלויות חדשות נדרשות:
- `prismjs` או `shiki` - Syntax highlighting
- `react-diff-view` - Diff viewer
- `react-markdown` - Markdown parsing משופר
- `d3` או `vis.js` - File dependency graph
- `clipboard-polyfill` - Copy to clipboard

### שיקולי ביצועים:
- Virtualization לשיחות ארוכות
- Lazy loading של הדגשת תחביר
- Debouncing בחיפוש
- Caching של גרפים

### נגישות:
- תמיכה במקלדת מלאה
- Screen reader support
- High contrast mode
- כיוון RTL לעברית
